## Getting Started

### Prerequisites

- Node.js 18+ and npm/yarn

### Installation

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

### PWA Icons

Replace the placeholder icon files in `public/` with actual images:
- `pwa-192x192.png` - 192x192px icon
- `pwa-512x512.png` - 512x512px icon  
- `apple-touch-icon.png` - 180x180px for iOS

The PWA manifest is configured in `vite.config.js`.

## Development

The app runs on `http://localhost:3000` by default.

### Key Routes

- `/` - Landing page
- `/how-it-works` - How it works guide
- `/start-test` - Start vision test onboarding
- `/eye-scan` - AI eye scanning interface
- `/mini-games` - Interactive vision tests
- `/results` - Test results and insights
- `/eyewear` - Eyewear recommendations
- `/ar-try-on` - AR glasses try-on
- `/profile` - User profile and history

## Building for Production

```bash
npm run build
```

The production build will be in the `dist/` directory, ready to deploy to any static hosting service.

## Deployment

The app can be deployed to:
- Vercel
- Netlify
- GitHub Pages
- Any static hosting service

Make sure to serve over HTTPS for PWA features to work properly.
